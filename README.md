# Git-e

Some useful cli tools for git

## Features

- [ ]  Project management
    - [ ]  Add project
        - [ ]  Set email
        - [ ]  Set directory
        - [ ]  Create ssh key (and ask for setting it on remote side)
        - [ ]  Set origin
        - [ ]  Set default branch (as I understand it can be fetched, but user should approve it)
        - [ ]  Set task branch template (like <prefix>/<task number>-few-words-of-description)
        - [ ]  Set prefixes (feature, hotfix, bugfix)
    - [ ]  Modify project
    - [ ]  Delete project
    - [ ]  Read project settings
- [ ]  Task management
    - [ ]  Create task branch
    - [ ]  Delete old task branch
        - [ ]  Maybe when creating new
        - [ ]  Maybe add total clean command
    - [ ]  Commit changes (`git add -A; git commit`)
    - [ ]  Upload branch (`git push`)
    - [ ]  Pull changes (from default branch)
- [ ]  Utils
    - [ ]  Show commits
    - [ ]  Show branches
    - [ ]  Stash/unstash
    - [ ]  Forget/unforget
